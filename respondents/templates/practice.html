Verbs and Resources

Reference the Confirmit Docs page for complete information. Confirmit Docs exists inside every Horizons site e.g. for Nordic or the US site, etc...

Surveys: Everything concerning surveys. (GA)

 GET /v1/surveys/
 GET /v1/surveys/{surveyId}

Respondents: Everything concerning respondents. (GA)

 GET+POST+PATCH   /v1/surveys/{surveyId}/respondents
 GET+DELETE+PATCH /v1/surveys/{surveyId}/respondents/{respid}

Survey Data: Everything concerning survey data reading. (LA)

 GET /v1/surveys/{surveyId}/responses/schema
 GET /v1/surveys/{surveyId}/responses/data

Variables: Everything concerning respondent variables. (Preview?)

 GET+POST /v1/surveys/{surveyId}/variables
 GET      /v1/surveys/{surveyId}/variables/{name}

End Points

To reach end points, our clients will need an API professional user that has a client id and client secret tied to it. For the Surveys API, the pub.surveys scope must be defined when generating the client id and secret.

  • US
    1. https://idp.us.confirmit.com/identity/connect/token
    2. https://ws.us.confirmit.com/
  • EURO
    1. https://idp.euro.confirmit.com/identity/connect/token
    2. https://ws.euro.confirmit.com/
  • NORDIC
    1. https://idp.nordic.confirmit.com/identity/connect/token
    2. https://ws.nordic.confirmit.com/
  • TESTLAB
    1. https://idp.testlab.firmglobal.net/identity/connect/token
    2. https://ws.testlab.firmglobal.net
